# Tech Stack - Kamingo

Data: 30.11.2025

---

## Backend

- **Framework:** Symfony 7.2
- **Language:** PHP 8.2+
- **Database:** MySQL 8.0.44
- **ORM:** Doctrine
- **Authentication:** Symfony Security Component

---

## Frontend

- **Template Engine:** Twig

- **JavaScript:** Vanilla JS + Stimulus (Symfony UX)

## Testing
